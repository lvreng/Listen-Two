# 打包说明

## 打包命令

### 打包 Windows 安装程序（NSIS）
```bash
npm run build:win
```

### 打包便携版（Portable）
```bash
npm run build:win:dir
```

### 打包所有格式
```bash
npm run build
```

## 输出位置

打包后的文件会输出到 `dist/` 目录：
- **安装程序**: `Listen Two-1.0.0-x64.exe` (NSIS 安装程序)
- **便携版**: `Listen Two-1.0.0-portable.exe` (无需安装，直接运行)

## 优化说明

### 启动速度优化
1. **延迟显示窗口**: 窗口在准备好后再显示，避免白屏
2. **禁用开发者工具**: 生产环境不加载开发者工具
3. **代码压缩**: 使用 asar 打包，压缩代码体积
4. **排除不必要文件**: 排除测试文件、文档等，减小体积

### 功能完整性
- ✅ 所有核心功能已包含
- ✅ 所有资源文件已包含
- ✅ 所有依赖已打包

## 注意事项

1. **首次打包**: 首次打包会下载 Electron 二进制文件，可能需要一些时间
2. **杀毒软件**: 某些杀毒软件可能会误报，这是正常现象（未签名应用）
3. **文件大小**: 打包后的文件约 100-150MB（包含 Electron 运行时）

## 分享建议

推荐分享**便携版**（Portable），因为：
- 无需安装，直接运行
- 可以放在 U 盘等移动设备中
- 不会在系统中留下注册表项
